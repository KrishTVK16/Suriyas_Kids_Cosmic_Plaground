A beautiful 15-page online kids magazine website with a magical cosmic theme, built with HTML, CSS, JavaScript, Bootstrap, and AJAX.

## Features

- **15 Unique Pages**: Each with its own layout while maintaining a consistent cosmic theme
- **Dynamic Content Loading**: All content loads via AJAX without page reloads
- **Beautiful UI**: Glassmorphism and Neomorphism design styles
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: Hover animations, parallax scrolling, twinkling stars, glowing buttons

## Theme

- **Colors**: Deep midnight blue, nebula purple, starlight white, neon pink & turquoise accents
- **Visual Motifs**: Floating planets, stars, rockets, comets, whimsical clouds
- **Typography**: Baloo 2 / Fredoka One for headings, Poppins / Inter for body text

## How to Use

1. Open `index.html` in a web browser
2. Navigate through the pages using the navigation menu
3. All content loads dynamically via AJAX from JSON files in the `data/` folder

## Technologies Used

- HTML5
- CSS3 (with Glassmorphism & Neomorphism)
- JavaScript (ES6+)
- Bootstrap 5.3.0
- AJAX (XMLHttpRequest)

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
